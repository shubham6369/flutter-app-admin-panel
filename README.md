# JEE NEET Pulse | Admin Panel

A premium, high-performance administrative dashboard built with Next.js 14, TypeScript, and Firebase.

## Features

- **Advanced Analytics**: Real-time revenue tracking, student engagement, and course performance charts.
- **Curriculum Management**: Complex course structures with sections and multiple lecture types (Video, PDF, YouTube).
- **Live Streaming**: Schedule and manage live YouTube lectures directly from the dashboard.
- **Sales Tracking**: Complete transaction history with student details and payment status.
- **Premium UI**: Dark-themed glassmorphism design for a professional look.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Database**: Firebase Firestore
- **Authentication**: Firebase Auth
- **Icons**: Lucide React
- **Charts**: Recharts
- **Deployment**: Vercel

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Firebase Constants

The app is already configured with the JEE NEET Pulse Firebase credentials in `src/lib/firebase.ts`.

## Deployment to Vercel

Push this code to GitHub and import it into Vercel. The project is already optimized for Vercel deployment.
